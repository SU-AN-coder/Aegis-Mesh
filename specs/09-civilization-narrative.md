# Civilization Narrative

## 1. 定位句

Aegis Mesh 不是“更高效的战斗插件”，而是一个让联盟在高冲突宇宙里仍能维持互助秩序的文明基础设施。

## 2. 三个叙事支点

1. 互助条约（Mutual Aid Treaty）
- 玩家不是孤立个体，求救、响应、赔付通过规则形成可信协作。

2. 文明底线（Civil Baseline）
- ceasefire、civilianProtection、审计裁决将“能打”与“该不该打”分离。

3. 可验证公正（Auditable Justice）
- 证据、裁决、赔付全部可追溯，不靠口头信誉。

## 3. 视频讲述模板（5-8 分钟）

1. 世界问题（30s）
- 运输与求救缺乏可信协作，冲突导致信任塌缩。

2. 工具箱方案（60s）
- ACL 权限、RoutePass、Distress、Incident、Policy 四条链路。

3. 实机闭环（3-4min）
- 靠近 gate -> sponsor 通行
- 遭袭求救 -> 响应接单
- KillMail 取证 -> 审计裁决 -> insurer 赔付
- 策略切换后系统行为变化

4. 文明立意（60s）
- 规则不仅提高效率，更降低背刺成本，建立联盟间合作边界。

5. 未来扩展（30s）
- 跨联盟保险池、信誉层、条约自动执行。

## 4. 项目简介模板

Aegis Mesh is a live-world security and logistics protocol for EVE Frontier. It combines sponsored routing, distress response, incident adjudication, and policy governance into one auditable loop. Instead of optimizing conflict only, it establishes mutual-aid treaties and civil baselines through smart contracts, making cooperation enforceable and justice verifiable in a hostile universe.

## 5. 评委问答准备

1. 为什么不是普通看板？
- 因为我们直接改变 gate/turret 运行规则，并提供链上裁决与赔付执行。

2. Live 怎么证明？
- 提供 Stillness 实机视频、官方数据源绑定日志、链上 digest 对照。

3. 去中心化体现在哪？
- 单 operator 已升级为 ACL 角色体系，敏感路径支持双角色审批。

4. 文明主题怎么落地？
- 通过条约豁免、平民保护、可审计裁决，把“互助与底线”写进协议。
